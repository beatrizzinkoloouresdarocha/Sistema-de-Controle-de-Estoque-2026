# 📦 Sistema de Controle de Estoque

Projeto desenvolvido em **Python** para gerenciamento de estoque com persistência de dados em **JSON**, utilizando **Git** e **GitHub** para versionamento.

---

## 📖 Sobre o Projeto

O Sistema de Controle de Estoque foi desenvolvido com o objetivo de praticar lógica de programação, organização de código em módulos e manipulação de arquivos JSON.

O sistema permite cadastrar, listar, buscar, alterar e excluir produtos, armazenando as informações em um arquivo `estoque.json`, garantindo que os dados permaneçam salvos mesmo após o encerramento do programa.

---

## 🚀 Funcionalidades

* Cadastro de produtos
* Listagem de produtos
* Busca de produtos
* Alteração de produtos
* Exclusão de produtos
* Geração automática de ID
* Salvamento dos dados em arquivo JSON
* Interface via terminal

## ▶️ Como Executar

1. Clone este repositório:


2. Entre na pasta do projeto:

```bash

```

3. Execute o programa:

